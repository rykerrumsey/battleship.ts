# Battleship.ts
---
## Requirements
* [Node.js 14.11.0](https://nodejs.com)
* Fullscreen linux terminal... not tested on Windows Shell

## Install
1. Check Node and Npm versions
```
node -v
v14.11.0
```
```
npm -v
6.14.8
```
2. Install application dependencies
```
npm i
```
3. Test the application
```
npm t
```
4. Run Battleship.ts
```
npm start
```
